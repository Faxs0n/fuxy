# fuxy

This command-line utility simplifies development environment setup by generating customizable lists of targets and vulnerabilities.


How to Use:




1.Run the Script:


    Make sure you have Python 3 installed. Execute the script from your terminal:


     


     python fuxy.py


How to Use the FUXY Tool




2.Input Target Profile:

Run the Script:

3.Set Options:


    You will specify:


        Minimum Password Length (default: 6)

        Output Filename (default: fuxy_wordlist.txt)





4.Preview Generated Passwords:


    After generating the passwords, you can choose to preview the first 20 generated passwords.

Set Options:


        Choose the Minimum and Maximum Password Length.


        Select mutation options:


            Common word combos


            Keyboard walks

            Prefix mutations


            Separator combos





Generate Passwords:


    Click the "▶ GENERATE" button to create a list of generated passwords based on the input and selected options.





Save the Password List:


    Click the "↓ SAVE .TXT" button to save the generated passwords to a text file.





Clear Fields:


    If you want to start over, click the "✖ CLEAR" button to reset all fields.
