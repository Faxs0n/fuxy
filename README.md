# fuxy
This command-line utility simplifies development environment setup by generating customizable lists of targets and vulnerabilities.

How to Use the FUXY Tool

Run the Script:
    Ensure you have Python 3 installed. Execute the script from your terminal or double-click it if you're on a compatible system.

bash

    python fuxy.py

Enter Profile Information:
    In the GUI, fill out the required fields, including:
        First Name
        Nickname / Alias
        Date of Birth (format: dd/mm/yyyy)
        Partner's Name
        Pet's Name
        Child's Name
        Company
        School
        Hobby
        Favourite Sport
        Favourite Word
        Favourite Color
        Car / Brand
        Favourite Artist
        Username
        Email Username
        Birth City
        Phone Digits (last 4)

Set Options:
        Choose the Minimum and Maximum Password Length.
        Select mutation options:
            Common word combos
            Keyboard walks
            Reversed words
            Doubled words
            Prefix mutations
            Separator combos

Generate Passwords:
    Click the "▶ GENERATE" button to create a list of generated passwords based on the input and selected options.

Save the Password List:
    Click the "↓ SAVE .TXT" button to save the generated passwords to a text file.

Clear Fields:
    If you want to start over, click the "✖ CLEAR" button to reset all fields.

Important Note:
    This tool is intended for authorized testing only. Use it responsibly and ensure you have permission to test any systems.

This concise guide will help users effectively understand how to operate the FUXY v2.0 tool while ensuring a clear representation of its functionalities.
