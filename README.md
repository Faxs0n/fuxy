# fuxy
This command-line utility simplifies development environment setup by generating customizable lists of targets and vulnerabilities.
How to Use:

    1.Run the Script:
    Make sure you have Python 3 installed. Execute the script from your terminal:
    $python fuxy.py

    2.Input Target Profile:
    You will be prompted to enter the following details:
        First Name
        Nickname / Alias
        Date of Birth (format: dd/mm/yyyy or leave blank)
        Partner's Name
        Pet's Name
        Child's Name
        Company / School
        Hobby / Sport
        Favourite Word

    3.Set Options:
    You will specify:
        Minimum Password Length (default: 6)
        Maximum Password Length (default: 20)
        Output Filename (default: fuxy_wordlist.txt)

    4.Preview Generated Passwords:
    After generating the passwords, you can choose to preview the first 20 generated passwords.

    5.Important Note:
    FUXY is intended for use only on systems you own or where you have explicit authorization to conduct testing.

Example Command Line Execution:

bash

python fuxy.py

Follow the prompts to complete the process and generate your custom wordlist.
